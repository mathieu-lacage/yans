Release 0.1

The first release: release 0.1. This release can be considered a prototype since it does not implement all the features we want but we are working on adding all the missing pieces. So far, this release implements:

  • a UDP/IPv4 stack without multicast
  • a nice packet API
  • a complete arp module
  • an empty ethernet module which uses the arp module to perform address translation between IPv4 addresses and MAC addresses.
  • libpcap dump support at the network interface level
  • a sample application with a simple udp source and sink pair.
  • the framework to provide a POSIX environment (this is far from the real thing for now).
